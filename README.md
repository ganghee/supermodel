클린 아키텍처에서 dart model을 만드는 플러그인 입니다.

Entity의 Response모델 UseCase의 Dto모델 Presenter의 Vo 모델을 한 번에 생성합니다.

파일하나에 객체 모델 하나씩 분리 여부를 설정할 수 있으며 3개의 모델을 원하는 위치의 폴더에 생성할 수 있습니다.
